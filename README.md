# 📊 Student Performance Prediction (Machine Learning Project)

## 🔍 Overview

This project predicts student marks based on study hours using a Machine Learning algorithm called **Linear Regression**. It demonstrates how data can be used to make predictions and visualize patterns.

---

## 🚀 Features

* Predicts marks based on study hours
* Uses Linear Regression model
* Accepts user input for prediction
* Displays model accuracy
* Visualizes data using graphs

---

## 🧠 Tech Stack

* Python
* NumPy
* Scikit-learn
* Matplotlib

---

## ⚙️ How It Works

1. Dataset is created with study hours and marks
2. Data is split into training and testing sets
3. Linear Regression model is trained
4. Model predicts marks for given input
5. Results are displayed along with a graph

---

## ▶️ How to Run

### Step 1: Install dependencies

pip install numpy scikit-learn matplotlib

### Step 2: Run the project

python student_model.py
